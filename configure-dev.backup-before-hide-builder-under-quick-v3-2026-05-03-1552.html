Important Install Instructions
Ultra MAX has two separate parts:

1. Addon URL
After clicking Generate Final Setup, copy the final setup URL and use it to add/install Ultra MAX in Nuvio or Stremio.

2. Nuvio Collections JSON
Click Export Nuvio JSON to download your collections file. Then open Nuvio, go to the Collections page, and import that JSON file there.

Simple rule:
Addon URL = catalogs. Nuvio JSON = folders and collections.
